# ADRs — Desafio 01

Coloque aqui os Architecture Decision Records na nomenclatura:

`ADR-NNN-titulo-curto.md`

Use o template em `../../shared/templates/ADR.md.tpl` (se existir) ou siga o formato:

```markdown
# ADR-001: Título da Decisão

## Status
Proposta | Aceita | Rejeitada | Depreciada

## Contexto
O que motivou esta decisão?

## Decisão
O que foi decidido?

## Consequências
- Positivas: ...
- Negativas: ...
- Riscos: ...
```
